# Leaf-diseased-detection
Leaf Disease Detection using Machine Learning
# Installation


### Run Locally

Clone the Repositories

```bash
  git clone (#link of repositories)
```

Go to the project directory

```bash
  cd flower-detection
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python leaf.py
```
